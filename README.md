This repository contains my personal solutions to the problems from the [CSES Problem Set](https://cses.fi/problemset/), a collection of competitive programming tasks maintained by the University of Helsinki.

The solutions are organized into sub-directories corresponding to the problem set categories. Each directory contains a specific `README.md` with problem details and the solution files.
